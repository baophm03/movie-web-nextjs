## Movie App
Web xem thông tin phim, xây dựng bằng Next.js, sử dụng dữ liệu từ [The Movie Database (TMDB)](https://www.themoviedb.org/).

## Getting Started
Clone repo
```
git clone https://github.com/baophm03/movie-web-nextjs
cd movie-app
```
Install node_modules
```
npm install
```
Run:
```
npm run dev
```
